# bilibili-helper

A Node.js Toolkit for Bilibili live service

## Installation

```bash
npm run build
npm run test:room
```

## Tip
* 本项目最初源码由[pandaGao](https://github.com/pandaGao)开发的[bilibili-live](https://github.com/pandaGao/bilibili-live)生成
* 由于老项目已无法正常使用，造成新版BiliBili live使用的问题，本仓库代码进行维护更新
* 感谢pandaGao的贡献
